## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development

$ docker-compose up
$ npm run start

# watch mode
$ docker-compose up
$ npm run start:dev

# production mode
$ docker-compose up
$ npm run start:prod
```
